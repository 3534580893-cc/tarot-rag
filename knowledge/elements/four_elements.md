# 四元素与塔罗

塔罗的四个花色分别对应四大元素，每种元素代表生命的不同维度。

## 火元素 — 权杖（Wands）
- **本质：** 意志、激情、创造力、行动
- **关键词：** 热情、勇气、灵感、进取心、冒险、变革
- **阴暗面：** 冲动、愤怒、焦虑、过度自信、注意力分散
- **对应星座：** 白羊座、狮子座、射手座
- **生活领域：** 事业、创业、激情项目、灵性追求、个人力量
- **问题类型：** 职业发展、创意工作、动力与激情、竞争

## 水元素 — 圣杯（Cups）
- **本质：** 情感、直觉、关系、潜意识
- **关键词：** 爱情、同理心、梦境、想象力、治愈、直觉
- **阴暗面：** 过度情绪化、依赖、幻想、逃避、敏感脆弱
- **对应星座：** 巨蟹座、天蝎座、双鱼座
- **生活领域：** 爱情、友谊、家庭、情感健康、艺术创作
- **问题类型：** 感情关系、情绪状态、内心感受、家庭

## 风元素 — 宝剑（Swords）
- **本质：** 思想、沟通、冲突、真相
- **关键词：** 清晰、逻辑、决断、真相、挑战、智识
- **阴暗面：** 过度分析、冷酷无情、冲突制造、焦虑担忧
- **对应星座：** 双子座、天秤座、水瓶座
- **生活领域：** 沟通、思考、决策、冲突、法律事务、学习
- **问题类型：** 争论与冲突、决策困难、沟通问题、思维模式

## 土元素 — 星币（Pentacles）
- **本质：** 物质、稳定、实际、健康
- **关键词：** 繁荣、稳定、脚踏实地、勤劳、财富、可靠
- **阴暗面：** 固执、物质主义、停滞、过于保守、贪婪
- **对应星座：** 金牛座、处女座、摩羯座
- **生活领域：** 金钱、健康、工作、家庭稳定、身体层面
- **问题类型：** 财务状况、工作稳定、健康问题、物质安全

## 元素互动关系
- **火+风：** 相互增强，行动力强劲
- **水+土：** 相互滋养，情感与物质的稳定
- **火+水：** 相互对立，激情与情感的张力
- **风+土：** 相互对立，思想与实践的矛盾
